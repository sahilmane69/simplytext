# Product Requirements

SimplyText is strictly a text-only messaging app.

Only plain text messages are allowed. Do not implement or plan support for images, videos, voice notes, file sharing, GIFs, stickers, audio/video calls, or status/stories.

Keep the existing architecture focused on text chat only. Avoid adding media capture, upload, picker, attachment, preview, playback, or calling dependencies.
