# Product Requirements

SimplyText is strictly a text-only messaging app.

Only plain text messages are allowed. Do not implement or plan support for message images, videos, voice notes, file sharing, GIFs, stickers, audio/video calls, or status/stories.

Profile creation may include a username, optional bio, and avatar/profile photo. Keep media support scoped to profile identity only; do not add media capture, upload, picker, attachment, preview, playback, or calling dependencies for chats or groups.
